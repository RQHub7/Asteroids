# Asteroids - V 0.1
#### Python 3.13.12 ... updates to come 

A simple version of the Asteroids arcade game. Shoot asteroids and survive!

![alt text](image.png)

### <u>Features</u>
* Shoot up to 3 shots per second with your spacebar
* move with w, a, s & d keys
* Asteroids will explode into smaller ones
* Game closes on death
* Venture off the window if you dare!

### <U>Requirements</u>
* Builds off of pygame 2.6.1 & random

### <u>Future updates</u>
* Score Tracking
* Collision Improvements
